# Hospital Management System (SQL)

A normalized relational database system designed for managing hospital operations, built using MySQL as part of a DBMS course project.

## Database Structure
- 15+ interconnected tables including Patient, Doctor, Appointment, Billing, Insurance, and Medical Records
- Primary and foreign key constraints for data integrity
- Normalized schema design

## SQL Features Used
- Complex JOINs across multiple tables
- Subqueries and nested queries
- Aggregate functions (COUNT, SUM, AVG)
- Set operations
- Views and stored procedures

## Technologies Used
- MySQL
- SQL
